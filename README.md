# 使用说明
## 使用环境
需要python3环境
## 使用步骤
- 进入目录
- windows执行以下命令
```shell
python sendemail.py
```
- Linux执行以下命令
```shell
python3 sendemail.py
```
## 备注
- 授权码需要在登录邮箱在设置里开启
- 发送前需要确认依赖的模块是否安装
- windows执行以下命令
```shell
pip install subprocess smtplib email os
```
- Linux执行以下命令
```shell
pip3 install subprocess smtplib email os
```
